# Database

Contiene los recursos de base de datos de PhysicsPlay para Supabase PostgreSQL.

- `scripts/`: creacion de tablas, datos iniciales y migraciones.
- `procedures/`: procedimientos almacenados o funciones de consulta.
- `backups/`: respaldos de desarrollo o ejemplos de restauracion.

El backend Spring Boot tambien incluye una migracion Flyway equivalente en `backend/src/main/resources/db/migration/V1__init_hu007.sql`.
