# Models

Representan las entidades principales de PhysicsPlay.

Modelos previstos para HU007:

- `Student`: estudiante que resuelve retos.
- `PhysicsTopic`: tema de Fisica Dinamica.
- `Challenge`: reto gamificado.
- `Exercise`: ejercicio interactivo.
- `Attempt`: intento de resolucion.
- `Progress`: progreso academico por tema.
- `Recommendation`: actividad sugerida para refuerzo.
- `Badge`: insignia obtenida por logros o constancia.
