# Services

Contienen las reglas de negocio de PhysicsPlay.

Servicios previstos para HU007:

- `ChallengeService`: administra retos, ejercicios y niveles.
- `ScoringService`: calcula puntaje, progreso e insignias.
- `DifficultyDetectionService`: detecta errores frecuentes y bajo desempeno.
- `RecommendationService`: propone ejercicios de refuerzo.
- `LearningAnalyticsService`: prepara indicadores para analisis con IA.
