# Controllers

Reciben las solicitudes del frontend y coordinan la respuesta del sistema.

Controladores previstos para HU007:

- `ChallengesController`: lista retos disponibles y detalle de ejercicios.
- `AttemptsController`: recibe respuestas, tiempos e intentos.
- `ProgressController`: muestra avance, puntaje, logros y temas con dificultad.
- `RecommendationsController`: entrega ejercicios de refuerzo personalizados.
