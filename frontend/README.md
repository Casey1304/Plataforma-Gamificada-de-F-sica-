# Frontend

Contiene las interfaces de usuario de PhysicsPlay.

- `web-app/`: aplicacion web para estudiantes y docentes.
- `mobile-app/`: aplicacion movil futura para acceso desde celulares.

En la HU007, esta capa permite seleccionar retos de Fisica Dinamica, responder ejercicios, visualizar retroalimentacion inmediata y consultar el progreso del estudiante.
