# Docs

Documentacion del proyecto PhysicsPlay.

- `arquitectura/`: analisis funcional, arquitectura por capas y seleccion tecnologica.
- `historias-usuario/`: historias de usuario y criterios de aceptacion.
- `diagramas/`: diagramas Mermaid para arquitectura y flujo funcional.
- `physicsplay_postman_collection.json`: coleccion de endpoints principales, incluyendo IA predictiva con Gemini.
